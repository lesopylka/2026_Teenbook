## Примеры запросов

```
SELECT ?item ?itemLabel ?description WHERE {
    VALUES ?item {
    wd:Q37226
    wd:Q48282
    wd:Q159334
    wd:Q491
    wd:Q240126
    wd:Q180684
    wd:Q11024
    }

  SERVICE wikibase:label { bd:serviceParam wikibase:language "ru,en". }
  OPTIONAL {
    ?item schema:description ?description
    FILTER(LANG(?description) = "ru")
  }
}
```

## Процесс работы

В результате выполнения SPARQL-запроса удалось получить ограниченное число прямых связей между понятиями из выбранной предметной области. Полученные связи оказались недостаточными для построения содержательной схемы школьных отношений, поэтому итоговая онтология была дополнена и структурирована вручную на основе анализа темы.
